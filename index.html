• Proyecto Winform ABM (C# .Net)

Mi primer proyecto que me gustaría compartir es una aplicación de gestión de base de datos (ABM) que desarrollé junto con un compañero durante nuestros días en la facultad. Este proyecto fue un verdadero desafío para nosotros, ya que estábamos aprendiendo el lenguaje C# .NET. Sin embargo, nos lanzamos sin miedo y dimos lo mejor de nosotros. La aplicación consiste en una ABM que administra Artículos, Marcas y Categorías. Permite realizar operaciones como modificar, agregar y eliminar elementos, así como realizar filtros de búsqueda y ver detalles de cada artículo. Este proyecto representa nuestros primeros pasos en el mundo del desarrollo de software y nos brindó una valiosa experiencia en la aplicación práctica de los conceptos que estábamos aprendiendo en la universidad.
Podes encontrar el código fuente en nuestro repositorio de GitHub: github.com/Francoaguado08/-Tp-Winform-Equipo-6

• Proyecto Sockets (C++)

Otro proyecto que quiero destacar es uno que desarrollé de forma autodidacta utilizando programación de sockets. Este proyecto me permitió crear una relación entre un servidor y un cliente para simular transacciones financieras. La funcionalidad principal consiste en que el cliente ingresa un monto, un número de tarjeta y un número de seguridad, luego el servidor verifica si estos datos se encuentran dentro de un rango determinado para determinar la validez de la tarjeta. Este proyecto representó un desafío significativo para mí, ya que nunca había trabajado con sockets antes y fue una de mis primeras incursiones en el aprendizaje autodidacta. Por eso, tiene un valor sentimental especial para mí.
Si te interesa, echale un vistazo en mi repositorio de GitHub: github.com/SimonAltgelt/sockets-priv
